# IP Geo Follow

让 Chrome / Edge 浏览器 GPS 定位自动跟随当前公网 IP 地址位置的扩展。

## 中文使用说明

1. 打开 Chrome `chrome://extensions/` 或 Edge `edge://extensions/`。
2. 开启“开发者模式”或“开发人员模式”。
3. 点击“加载已解压的扩展程序”或“加载解压缩的扩展”。
4. 选择这个 `ip-geo-follow` 文件夹。
5. 点击扩展图标查看当前 IP、位置、经纬度、更新时间。
6. 出口 IP 变化后，点击“立即刷新”。
7. 如果目标网页之前已经请求过定位，刷新目标网页一次。

注意：这个扩展只修改浏览器 GPS 定位结果，不会修改你的真实 IP。真实 IP 取决于你的 VPN、代理或 VPS 出口。

## English Usage

1. Open Chrome `chrome://extensions/` or Edge `edge://extensions/`.
2. Enable Developer mode.
3. Click "Load unpacked".
4. Select this `ip-geo-follow` folder.
5. Click the extension icon to view the current IP, location, coordinates, and update time.
6. When your exit IP changes, click "立即刷新".
7. If the target page requested location before injection, reload the target page once.

Note: This extension only changes the browser Geolocation API result. It does not change your real IP address. Your real IP depends on your VPN, proxy, or VPS exit node.
